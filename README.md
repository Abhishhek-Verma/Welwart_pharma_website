# Welwart Pharmaceuticals Website

A professional, responsive, and SEO-optimized corporate website for **Welwart Pharmaceuticals Pvt. Ltd.**

## Overview
This project is a premium, medical-grade web presence built to highlight the company's product portfolios, quality assurance timelines, and interactive contact systems. It features a modern, high-performance aesthetic using a soft teal and slate palette.

## Technologies Used
- **HTML5** for semantic structure
- **CSS3** (Vanilla) for custom styling and dynamic hover effects
- **Vanilla JavaScript** for interactive logic and state management
- **GSAP (GreenSock Animation Platform)** for smooth, advanced scroll and UI animations

## Features
- **Responsive Layout**: Fully adaptive design that works seamlessly across mobile, tablet, and desktop devices.
- **Advanced Animations**: GSAP-powered scroll reveal effects and custom cursor interactions.
- **Particle Hero Background**: Dynamic, interactive particle effects in the hero section for a premium aesthetic.
- **Product Portfolio**: Organized product showcase with filtering capabilities.
- **Quality Assurance Timeline**: Interactive timeline highlighting the company's dedication to quality.
- **Contact System**: Fully functional and styled contact form with real-time feedback.

## Running Locally
To view the website locally, simply start a local static file server in the root directory.

For example, using `live-server` or `serve` via Node.js:
```bash
npx live-server
# or
npx serve .
```

Then, open your browser to the URL provided in the terminal.
